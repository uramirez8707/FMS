Detailed Description

Module to hold pointer and stack data for use in MPP modules.

Makes stack and pointer data publicly available from mpp_data_mpi.inc or mpp_data_nocomm.inc for use in MPP modules. This module is mainly for internal use within mpp_mod and mpp_domains_mod .
