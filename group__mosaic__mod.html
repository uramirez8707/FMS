Detailed Description

Implements some utility routines to read mosaic information.

Author
Zhi Liang Implements some utility routines to read mosaic information. The information includes number of tiles and contacts in the mosaic, mosaic grid resolution of each tile, mosaic contact information, mosaic exchange grid information. Each routine will call a C-version routine to get these information.
